# The Queries to run after starting the application :: 

-----

NOTE : Make sure that you have a parking_lot_db database created in your local mysql server before starting the application and run these queries only after starting the application.

## QUERY 1: To update the timestamp of parking_lot Table

```sql
ALTER TABLE parking_lot MODIFY created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP;
```

---

---

## QUERY 2: To initialize a Parking Lot

```sql
INSERT INTO parking_lot (lot_name, location, total_floors, total_spots)
VALUES ('Downtown Parking', 'Main Street, City Center', 3, 300);
```

--- 

## QUERY 3: Get the parking lot ID for next inserts

```sql
SET @parking_lot_id = LAST_INSERT_ID();
```
